# Phigros RKS 计算器 from ADB
## 介绍
通过 ADB 生成的 Android Backup (\*.ab) 文件计算 Ranking Score，并输出 png 图片。
## 环境和依赖
* Java (class file version > 52)
* Python 3
    * pyDes
    * lxml
	* requests
	* pillow
## 使用
直接
```
python main.py
```

> Windows 可能存在杂七杂八的问题，未经测试。  
> 我写了一坨大粪，可能会非常慢，耐心等待。
