# Phigros RKS 计算器 from ADB
## 介绍
通过 ADB 生成的 Android Backup (*\*.ab) 文件计算 Ranking Score
## 环境和依赖
* Java
* Python 3
* pyDes
* lxml
## 使用
直接
```
python main.py
```
